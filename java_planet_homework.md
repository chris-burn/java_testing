# Single Class - Homework

Create a new directory 'planets'

Convert the following Ruby class into an equivalent Java class with tests.

```
class Planet

  def initialize(name, size)
    @name = name
    @size = size
  end

  def get_name()
    return @name
  end

  def get_size()
    return @size
  end

  def explode()
    return "Boom! #{@name} has exploded."
  end

end
```
